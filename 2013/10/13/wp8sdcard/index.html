WP8之读取SD卡

2013-10-13
tech

主题

  WP8之读取SD卡。

正文

  由于微软的限制，如果要访问SD卡的文件，需要在文件名后缀加一个‘y’，比如 ‘.txt’ 改成 ‘.txty’，‘.doc’ 改成 ‘.docy’，以此类推，但是RAR，PDF，FLV文件不需更改。目前支持的文件类型为：txty，xlsy，xlsxy，docy，docxy，ppty，pptxy，jpgy，jpegy，pngy，gify， bmpy， picy，zipy，mp4y，RAR，PDF，FLV。

  首先要打开访问SD卡的访问权限：

  在项目中依次打开Properties——WMAppManifest.xml，在‘功能’中选中‘ID_CAP_REMOVABLE_STORAGE’

其次以XML方式打开WMAppManifest.xml，在标签后加入文件访问权限：

  

  <FileTypeAssociation TaskID="_default" Name="GPX" NavUriFragment="fileToken=%s">

    <SupportedFileTypes>

      <FileType ContentType="application/txty">.txty</FileType>

      <FileType ContentType="application/xlsxy">.xlsxy</FileType>

      <FileType ContentType="application/xlsy">.xlsy</FileType>

      <FileType ContentType="application/docy">.docy</FileType>

      <FileType ContentType="application/docxy">.docxy</FileType>

      <FileType ContentType="application/ppty">.ppty</FileType>

      <FileType ContentType="application/pptxy">.pptxy</FileType>

      <FileType ContentType="application/jpgy">.jpgy</FileType>

      <FileType ContentType="application/jpegy">.jpegy</FileType>

      <FileType ContentType="application/pngy">.pngy</FileType>

      <FileType ContentType="application/gify">.gify</FileType>

      <FileType ContentType="application/bmpy">.bmpy</FileType>

      <FileType ContentType="application/picy">.picy</FileType>

      <FileType ContentType="application/zipy">.zipy</FileType>

      <FileType ContentType="application/mp4y">.mp4y</FileType>

      <FileType ContentType="application/rar">.rar</FileType>

      <FileType ContentType="application/pdf">.pdf</FileType>

      <FileType ContentType="application/flv">.flv</FileType>

    </SupportedFileTypes>

  </FileTypeAssociation>

</Extensions>

 

  要访问SD卡，首先要声明： private IEnumerable deviceList;

  获取SD卡信息：

  private async void getSDInfo()

{

    deviceList = await ExternalStorage.GetExternalStorageDevicesAsync();

    int i = 0;

    foreach (ExternalStorageDevice device in deviceList)

    {

        i++;

        ListBoxItem item = new ListBoxItem();

        item.Content = "SDcard Root : ...";

        item.DataContext = device.RootFolder;

        sdCardListBox.Items.Add(item);

    }

    if (i == 0)

    {

        textSDCardInfo.Text = "Can't find SD card ";

    }

    else

    {

        textSDCardInfo.Text = "";

    }

}

  访问目录下的文件夹和文件：

  IEnumerable folderList = await externalStorageDevice.GetFoldersAsync();

foreach (ExternalStorageFolder folder in folderList)

{

    ListBoxItem item = new ListBoxItem();

    item.Content = "Folder :   " + folder.Name;

    item.DataContext = folder;

    sdCardListBox.Items.Add(item);

}

IEnumerable<ExternalStorageFile> fileList = await externalStorageDevice.GetFilesAsync();

foreach (ExternalStorageFile file in fileList)

{

    ListBoxItem item = new ListBoxItem();

    item.Content = "File :   " + file.Name;

    item.DataContext = file;

    sdCardListBox.Items.Add(item);

}
